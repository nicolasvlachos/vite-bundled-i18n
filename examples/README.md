# Examples

This repo currently uses the root Vite app as the main live example.

See:

- `src/` for the demo application
- `vite.config.ts` for plugin integration
- `src/i18n.config.ts` for shared config
- `docs/examples.md` for SSR, vanilla, and data-file patterns

The demo app covers:

- global dictionary loading
- scope bundles
- locale switching
- generated types
- build output under `demo-dist/__i18n`
